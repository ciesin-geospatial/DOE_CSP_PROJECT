This directory contains JSON-formatted user-generated input files that are passed to SamBaseClass. 

They are in the format <model_name><timestamp>_inputs.json
Where model_name is a code for the desalination, solar thermal or financial model.
timestamp is in the format YYYY-MM-DD_hh-mm-ss
YYYY = year
MM = month
DD = day
hh = hour
mm = minute
ss = second

You may want to keep them as a record of the variable values chosen for the application. Feel free to delete all files in this directory except for this README.txt as needed.
